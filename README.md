## Overview

**Quantum Synthesis Matrix** is an experimental advanced custom indicator that models financial markets as a **9-dimensional quantum system**.

It applies concepts from quantum mechanics (wave functions, superposition, entanglement, tunneling, collapse, decoherence) to price action, volume, volatility, and multi-timeframe behavior to detect high-probability reversal and continuation zones.

The indicator visualizes:
- Probability density waves
- Wave function collapse points
- Quantum tunneling zones
- Entanglement across timeframes
- Monte Carlo path projections

Includes a comprehensive real-time **Quantum Dashboard** with dimensional analysis and predictive metrics.

> **Note**: This is a highly conceptual/experimental indicator blending physics analogies with market analysis. Optimized for **XAUUSD (Gold)**.

## Installation

1. Download `QuantumSynthesisMatrix.mq5`
2. Open MetaTrader 5 → File → Open Data Folder
3. Go to `MQL5/Indicators/`
4. Paste the `.mq5` file
5. Restart MT5 or refresh Navigator
6. Apply to chart (recommended: **XAUUSD**, H1–H4 timeframes)
